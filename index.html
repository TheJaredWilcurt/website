NW.js lets you call all Node.js modules directly from DOM and enables a new way of writing applications with all Web technologies.

  • New way of writing native applications with the web's most popular technologies.

  • Apps written in modern HTML5, CSS3, JS, and WebGL.

  • Complete support for Node.js APIs and all third party modules.

  • Call Node.js modules directly from the DOM.

  • Easy to package and distribute apps.

  • Available on Windows, OSX, & Linux.
